# go-spec-driven

> This entire repo has been written and revised by a *human* 

go-spec-driven is a template repository that aims to create a scaffold for _Golang Spec Driven AI Code Development_, aiming to enforce specification and requirements throughout continuous development and maintenance of a Golang application.
This repo does not contain any code, instead, its a compilation of documentation files that enforce pratices and patterns for a graceful ci/cd.
The folder structure is a suggestion, the only recommended one to be kept is the "docs/" folder and its contents.
